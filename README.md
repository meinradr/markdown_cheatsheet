_**or use both together**_

<details><summary> LISTS </summary>
<p>

#### Unordered

```markdown
* Item 1
* Item 2
  * Item 2a
  * Item 2b
```

* Item 1
* Item 2
  * Item 2a
  * Item 2b
  
---

#### Ordered

```markdown
1. Item 1
1. Item 2
1. Item 3
   1. Item 3a
   1. Item 3b
```

1. Item 1
1. Item 2
1. Item 3
   1. Item 3a
   1. Item 3b
  
---  

 #### Task List
 
 ```markdown
- [x] @mentions, #refs, [links](), **formatting**, and <del>tags</del> supported
- [x] list syntax required (any unordered or ordered list supported)
- [x] this is a complete item
- [ ] this is an incomplete item
 ```
 
- [x] @mentions, #refs, [links](), **formatting**, and <del>tags</del> supported
- [x] list syntax required (any unordered or ordered list supported)


<details><summary> BACKSLASH ESCAPE </summary>
<p>

```markdown
\*literal asterisks\*
```

\*literal asterisks\*

Markdown provides backslash escapes for the following characters:  
\\    backslash  
\`   backtick  
\*    asterisk  
\_    underscore  
\{\}   curly braces  
\[\]   square brackets  
\(\)   parentheses  
\#    hash mark  
\+    plus sign  
\-    minus sign (hyphen)  
\.     dot  
\!     exclamation mark  

</p>
</details>
